# macdosa — 天啓 천계 v2.0

> 天啓 WORKSPACE의 一員. CHEONGYE Protocol v2.0 적용.
> 최종 판단자: 찰스. AI는 제안, 찰스가 결정.

## 프로젝트 三極

### 天 (원리/목적)
AI 기반 맥북 최저가 분석 서비스. 구매자에게 정확한 시세 정보 제공.

### 地 (바탕/기술)
- Node.js + Express
- Claude API 연동 (맥북 가격 분석)
- 토스 결제 연동
- Render 배포

### 人 (매개/사용자)
맥북 구매를 고려하는 한국 사용자. 찰스가 운영.

## 一 (중심 선언)
"AI가 맥북 최저가를 분석해 구매자의 판단을 돕는다."

## 天啓 CHEONGYE Protocol v2.0

> 最終 판단자: 찰스. AI는 제안, 찰스가 결정. 用變不動本.

### 천계 6문 체크포인트 (작업 전 필수)
1. **一** — 이 변경이 최저가 분석 핵심 가치와 일치하는가?
2. **三** — 분석 엔진(天)·서버(地)·사용자 경험(人) 균형?
3. **合** — 입력→분석→결과 흐름이 끊기지 않는가?
4. **本** — 변해도 본(최저가 정확도)이 남는가?
5. **人** — 실제 구매자에게 닿는가? (보안: API키/결제 포함)
6. **次一** — 다음 개선 씨앗을 남기는가?

**2개 이상 흐리면 중단 후 찰스에게 보고.**
