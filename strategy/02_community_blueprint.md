> 2026-04-19 | 찰스 비전 기반 기획

---

## 一 (중심)
> "혼자 결정하던 맥북 구매를, 함께 결정하는 경험으로."
> 고객이 구매 후 사라지는 구조 → 구매 전후 모두 머무는 관계 공간

---

## 현재 구조의 문제
```
URL 입력 → AI 분석 → 결제(9,900 / 19,900) → [고객 사라짐]
                                                    ↑
                                            여기에 아무것도 없음
```

## 목표 구조
```
URL 입력 → AI 분석 → 결제 → [구매 여정 공유]
                              ↓
                    커뮤니티 진입 → 후기 작성 → 다음 구매자 신뢰 형성
                              ↓
                    재방문 → 친구 추천 → 글로벌 확장
```

### CARD 01 — 구매 인증 배지 시스템
**[MVP]**

**무엇인가**
실제 구매자가 맥북 구입 영수증(또는 사진)을 인증하면 "실구매자" 배지 부여.
배지 보유자의 후기는 상단 노출, 신뢰도 가중치 높음.

**작동 방식**
- 구매 인증: 영수증 이미지 업로드 → AI(Claude)가 모델명·구매가 파싱 → 자동 승인
- 배지 등급: 일반 후기(글) → 영수증 인증(+실구매자 배지) → 절약 금액 인증(+절약왕 배지)

**DB 구조**
```sql
CREATE TABLE purchase_proofs (
  id uuid PRIMARY KEY,
  user_id uuid REFERENCES users,
  macbook_model TEXT,
  purchase_price INT,
  saved_amount INT,
  proof_image_url TEXT,
  verified BOOLEAN DEFAULT false,
  verified_at TIMESTAMPTZ,
  created_at TIMESTAMPTZ DEFAULT now()
);
```

**사회적 증거 효과**
- "이 모델 지금까지 347명 구매 인증" 표시
- 평균 절약 금액 실시간 집계: "맥도사 사용자 평균 절약액: 28,400원"

**글로벌 고려**
- 영수증 언어 무관 (Claude가 파싱)
- 통화 자동 변환 (KRW / USD / JPY / EUR)

### CARD 02 — 절약 후기 피드
**[MVP]**

**무엇인가**
구매 완료 후 "나는 얼마 절약했다" 후기를 소셜 카드 형태로 공유하는 피드.
트위터/인스타 공유 가능한 OG 이미지 자동 생성.

**작동 방식**
```
구매 완료 → "후기 작성하기" CTA 노출 →
모델명 / 구매가 / 절약 금액 / 한 줄 소감 입력 →
"절약 카드" 이미지 자동 생성 (Canvas API) →
피드 등록 + SNS 공유
```

**절약 카드 예시**
```
┌────────────────────────────────┐
│  🍎 MacBook Pro 14" M4 Pro     │
│  구매가: 2,340,000원            │
│  맥도사 분석으로 절약: 💰 43,000원 │
│  "살까말까 3일 고민했는데 바로 샀어요" │
│  ─────────────────────────────  │
│  @macdosa.kr | 실구매자 인증 ✓   │
└────────────────────────────────┘
```

**사회적 증거 효과**
- 홈페이지 메인에 실시간 후기 스크롤 피드
- "오늘 절약된 총액: 1,240,000원" 라이브 카운터

**글로벌 고려**
- 언어 선택: KO / EN / JA
- 절약 카드 텍스트 i18n 지원

### CARD 03 — 실시간 딜 제보 게시판
**[MVP]**

**무엇인가**
PRO 사용자가 발견한 한정 특가, 원데이딜, 이벤트를 제보하면
커뮤니티 전체에 공유되는 "딜 레이더" 게시판.

**작동 방식**
- 제보 폼: URL + 가격 + 마감 시간 + 설명
- AI 검증: Claude가 URL 접속 → 실제 가격 확인 → 허위 제보 필터
- 검증 완료 → 피드 등록 + PRO 구독자 푸시 알림

**제보자 보상**
- 검증된 제보 1건 = "딜헌터" 포인트 +10
- 포인트 100점 = PRO 1개월 무료 (재방문 인센티브)

**게시판 구조**
```
[딜 레이더] 지금 핫한 맥북 특가 ← 실시간
├── 🔥 [종료 2시간 전] 하이마트 맥북에어 15 M3 - 1,389,000원
│   └── 검증 완료 ✓ | 제보자: 맥덕후킴 | 클릭 1,240회
├── ⏰ [내일 마감] 쿠팡 로켓직구 맥북프로 14 M4 - $1,599
└── 💤 [종료] 애플 리퍼 맥북에어 13 - 1,150,000원
```

**사회적 증거 효과**
- 딜 클릭수 / 구매 전환 추적 → "이 딜로 32명이 구매했어요" 표시
- 딜헌터 TOP10 리더보드

**글로벌 고려**
- 지역 태그 필수: [KR] [US] [JP] [EU]
- 환율 자동 변환 표시

### CARD 04 — 구매 토론방 (모델별 채팅)
**[MVP]**

**무엇인가**
"지금 살까 기다릴까?" 고민하는 사람들이 모이는 모델별 토론 채널.
각 맥북 모델마다 전용 채팅방 존재.

**채널 구조**
```
#맥북에어-13-M3
#맥북에어-15-M3
#맥북프로-14-M4
#맥북프로-16-M4
#맥미니-M4
#자유토론
#구매완료-인증 ← 여기서 후기로 자연 연결
```

**AI 도우미 통합**
- 봇 "@맥도사봇"이 상주
- 사용자가 URL 던지면 → 자동 분석 결과 스레드에 첨부
- "지금 사는 게 맞나요?" → AI가 현재 가격 기준 판단 답변

**재방문 메커니즘**
- 내가 팔로우한 채널 새 글 → 이메일/푸시 알림
- "당신이 고민하던 맥북프로 14 가격이 5만원 떨어졌어요" 개인화 알림

**기술 구현**
- Supabase Realtime (WebSocket) → 무료 범위 내 처리 가능
- 메시지 보존: 90일 (FREE), 무제한 (PRO)

**글로벌 고려**
- 채널 언어 태그: [KR] [EN] [JA]
- 자동 번역 버튼 (Claude API)

### CARD 05 — 신뢰 점수 대시보드
**[MVP]**

**무엇인가**
맥도사 플랫폼 전체의 신뢰도를 숫자로 보여주는 "사회적 증거 허브".
홈페이지 상단/분석 결과 페이지에 항상 노출.

**표시 데이터**
```
┌─────────────────────────────────────┐
│  맥도사 신뢰 지표 (실시간)            │
│                                     │
│  🛒 총 분석 완료        12,847건     │
│  ✅ 실구매 인증         4,230명      │
│  💰 누적 절약 금액    ₩482,000,000  │
│  ⭐ 평균 만족도            4.8/5    │
│  🌍 사용 국가              23개국   │
└─────────────────────────────────────┘
```

**업데이트 주기**
- 분석 건수: 실시간
- 절약 금액: 구매 인증 시 즉시 반영
- 만족도: 후기 등록 시 재계산

**사회적 증거 효과**
- 신규 방문자: 숫자 보고 신뢰 형성
- 분석 결과 페이지: "이 모델 분석 847회 완료" 개별 표시

### CARD 06 — 맥북 구매 타임라인 (개인 구매 여정)
**[Phase 2]**

**무엇인가**
개인 사용자의 "맥북 구매 여정"을 기록하고 공유하는 타임라인.
고민 → 분석 → 구매 → 후기까지 스토리 형태로 보존.

**타임라인 예시**
```
📅 3월 12일 — 맥북프로 14 M4 첫 분석 (현재가: 2,590,000원)
📅 3월 19일 — 가격 알림 수신 (2,490,000원으로 하락)
📅 3월 21일 — 딜 제보 확인 (하이마트 10만원 추가 할인)
📅 3월 22일 — 구매 완료 🎉 (최종 2,340,000원 / 25만원 절약)
📅 3월 23일 — 후기 작성: "맥도사 없었으면 25만원 버릴 뻔"
```

**공유 기능**
- 타임라인 공유 → OG 이미지 + 링크 생성
- 커뮤니티 피드에 "구매 완료 스토리"로 노출

**재방문 효과**
- 타임라인 있는 사람 = 재방문율 3배 이상 (가설)
- 친구에게 공유 → 신규 유입 루프

### CARD 07 — 전문가 리뷰어 제도
**[Phase 2]**

**무엇인가**
맥북 리뷰 경험이 많은 사용자를 "공인 리뷰어"로 선정.
이들의 후기는 "전문가 검증" 배지와 함께 최상단 노출.

**선정 기준**
- 실구매 인증 3회 이상
- 후기 작성 5건 이상
- 커뮤니티 평점 4.5 이상

**혜택**
- PRO 구독 무료
- "맥도사 공인 리뷰어" 배지
- 직접 딜 제보 → 즉시 노출 (AI 검증 생략)

**수익 모델 연결**
- 미래: 리뷰어 전용 제휴 링크 → 구매 전환 시 수수료 지급
- 글로벌 확장 시 각 국가별 리뷰어 네트워크 구축

### CARD 08 — 글로벌 커뮤니티 구조
**[Phase 2]**

**언어/지역별 분리 구조**
```
macdosa.kr          ← 한국 (메인, 최초 론칭)
macdosa.kr/en       ← 영어권 (미국/캐나다/호주)
macdosa.kr/jp       ← 일본
macdosa.kr/global   ← 기타 영어 통합
```

**지역별 가격 특성 반영**
| 지역 | 특이사항 |
|------|---------|
| 한국 | 하이마트/쿠팡, 카드 할인 복잡, 부가세 환급 |
| 미국 | 교육 할인(unidays), 주세 차이, 리퍼 스토어 강력 |
| 일본 | 면세 구매, 엔저 활용, 포인트 중심 |
| 호주 | GST 환급, 환율 변동 중요 |

**글로벌 공통 피드**
- 전 세계 딜 제보 → 지역 필터 적용해서 각자에게 노출
- 글로벌 절약왕 리더보드 (USD 환산 기준)

**번역 전략**
- 커뮤니티 글: 수동 선택 번역 (Claude API, 클릭 시)
- UI/안내문: 정적 i18n (ko/en/ja 3개 우선)
- 딜 제보: 지역 태그 필수, 자동 분류

### CARD 09 — 관계 형성 메커니즘 (재방문 설계)
**[Phase 2]**

**재방문 트리거 5가지**

1. **가격 하락 알림** (이미 PRO 기능에 있음)
   - "당신이 분석한 맥북프로 14, 3만원 떨어졌어요"

2. **딜 레이더 알림**
   - "관심 모델 한정 특가 발견! 오늘만 10만원 할인"

3. **커뮤니티 반응 알림**
   - "내 후기에 댓글이 달렸어요 (맥덕후킴 님)"
   - "내 딜 제보를 47명이 클릭했어요"

4. **주간 가격 레포트** (PRO)
   - 매주 월요일 이메일: "지난주 맥북 가격 동향 + 이번 주 추천 딜"

5. **구매 기념일 알림**
   - "맥북 구매 1주년! 지금 팔면 얼마에 팔 수 있을까요?" → 재분석 유도

**소셜 그래프 형성**
- 팔로우 기능: 특정 리뷰어 팔로우 → 새 후기 알림
- 딜헌터 구독: 특정 딜헌터 팔로우
- 목표: 1인당 팔로우 3명 이상 = 이탈률 급감

### CARD 10 — 구매 전 AI 상담 (소셜 연결)
**[Phase 3]**

**무엇인가**
"지금 살까 기다릴까?" AI 판단 + 커뮤니티 의견을 동시에 보여주는 복합 상담 화면.

**구성**
```
[AI 판단] 현재가 기준 → "지금 구매 추천 (3달 최저가 근접)"
[커뮤니티 의견] 이 모델 토론방 최근 7일 → "78% 구매 추천 의견"
[실구매자 후기] 같은 모델 최신 후기 3개 → "평균 절약: 32,000원"
[딜 레이더] 이 모델 현재 활성 딜 → "하이마트 5% 추가 할인 중"
```

**효과**
- AI 혼자 판단 → AI + 사람 함께 판단으로 신뢰도 2배
- 커뮤니티 유입 자연 발생 (상담 결과 보다가 토론방 진입)

🏆 이번 달 절약왕 TOP 5

**글로벌 버전**
- 지역별 리더보드 + 글로벌 통합 (USD 환산)

## 기술 구현 우선순위

### MVP (Phase 1 — 현재 구조 위에 바로 추가)

| 기능 | 기술 | 난이도 | 우선순위 |
|------|------|--------|---------|
| 구매 인증 배지 (CARD 01) | Supabase Storage + Claude 파싱 | ★★☆ | 1순위 |
| 절약 후기 피드 (CARD 02) | Supabase DB + Canvas API | ★★☆ | 2순위 |
| 신뢰 점수 대시보드 (CARD 05) | Supabase 집계 쿼리 | ★☆☆ | 3순위 |
| 실시간 딜 제보 (CARD 03) | Supabase + Claude 검증 | ★★★ | 4순위 |
| 구매 토론방 (CARD 04) | Supabase Realtime | ★★★ | 5순위 |

**MVP 추가 DB 테이블**
```sql
-- 구매 인증
CREATE TABLE purchase_proofs (
  id uuid PRIMARY KEY,
  user_id uuid REFERENCES users,
  macbook_model TEXT,
  purchase_price INT,
  saved_amount INT,
  proof_image_url TEXT,
  verified BOOLEAN DEFAULT false,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- 후기 피드
CREATE TABLE reviews (
  id uuid PRIMARY KEY,
  user_id uuid REFERENCES users,
  proof_id uuid REFERENCES purchase_proofs,
  content TEXT,
  rating INT CHECK (rating BETWEEN 1 AND 5),
  share_card_url TEXT,
  likes INT DEFAULT 0,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- 딜 제보
CREATE TABLE deal_reports (
  id uuid PRIMARY KEY,
  reporter_id uuid REFERENCES users,
  macbook_model TEXT,
  deal_url TEXT,
  deal_price INT,
  expires_at TIMESTAMPTZ,
  verified BOOLEAN DEFAULT false,
  click_count INT DEFAULT 0,
  region TEXT DEFAULT 'KR',
  created_at TIMESTAMPTZ DEFAULT now()
);

-- 토론방 메시지
CREATE TABLE community_messages (
  id uuid PRIMARY KEY,
  channel TEXT,  -- 'macbook-pro-14-m4'
  user_id uuid REFERENCES users,
  content TEXT,
  lang TEXT DEFAULT 'ko',
  created_at TIMESTAMPTZ DEFAULT now()
);
```
